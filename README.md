# Projeto de Sistema Web MVC com CRUD e Autenticação

Este projeto é uma aplicação web desenvolvida utilizando C# MVC .NET com SQL Server, que inclui funcionalidades de CRUD (Create, Read, Update, Delete), autenticação de usuários, e manipulação de imagens. A estrutura segue o padrão MVC (Model-View-Controller), permitindo uma organização eficiente do código fonte.

## Funcionalidades Principais

- **CRUD:** Implementação de operações CRUD para gerenciar dados dos clientes, com tabelas relacionadas no banco de dados.
- **Autenticação de Usuários:** Sistema de login para garantir acesso seguro às funcionalidades da aplicação.
- **Inclusão de Imagens:** Capacidade de adicionar e exibir imagens de logo para cada cliente, proporcionando uma experiência visual personalizada.
- **Requisições AJAX:** Utilização de JavaScript para realizar requisições AJAX ao banco de dados, melhorando a interatividade e responsividade da aplicação.

## Tecnologias Utilizadas

- **C# MVC .NET:** Framework utilizado para o desenvolvimento da aplicação web, proporcionando uma arquitetura robusta e modular.
- **SQL Server:** Banco de dados utilizado para armazenar e gerenciar os dados dos clientes.
- **Stored Procedures:** Utilização de procedures SQL para otimizar e centralizar a lógica de negócio no banco de dados.
- **JavaScript:** Linguagem utilizada para a manipulação do DOM e realização de requisições AJAX.
- **Controllers e Managers:** Estruturação do código em controllers e managers em C#, seguindo boas práticas de desenvolvimento.

## Ferramentas

Este projeto foi uma oportunidade de utilização de diversas ferramentas de linguagens diferentes:

- **Desenvolvimento MVC:** Aprofundei meus conhecimentos na arquitetura MVC, compreendendo melhor a separação de responsabilidades entre os modelos, as views e os controllers.
- **SQL e Stored Procedures:** Aprendi a utilizar procedures SQL para otimizar o desempenho das consultas e centralizar a lógica de negócio no banco de dados.
- **Integração Frontend e Backend:** Utilizei JavaScript para criar uma experiência de usuário mais dinâmica, com requisições AJAX para manipular os dados de forma assíncrona.
- **Segurança:** Implementei um sistema de autenticação para garantir a segurança das informações dos usuários.
- **Boas Práticas de Desenvolvimento:** Segui boas práticas de desenvolvimento de software, como a utilização de controllers e managers para manter o código organizado e modular.
